- `00_overview`: Master vision and constitutional reference.

- `02_ux_design`: UX, screens, and Figma build guidance.
- `03_ai_engine`: AI engine, prompt architecture, and node catalog.
- `04_knowledge`: Quran, hadith, tafsir, graph, retrieval, ranking, and import specs.

- `08_operations`: Deployment and launch-readiness architecture.

- `90_archive`: Preserved source discussions, superseded V1 drafts, and research notes.
- `99_registers`: Inventory, canonical map, and correction register.
